# 快速入门

> wavEDA 建模仿真全流程概览，适合第一次使用的用户。

---

## 整体流程

```
几何建模 → 材料设置 → 端口设置 → 边界条件 → 求解配置 → 网格设置 → 仿真 → 后处理
```

## 详细步骤

### 1. 建模

| 步骤 | 操作 | 说明 |
|:----:|------|------|
| **①** | 导入/创建几何 | 导入 `.stp` 或直接建模，运用旋转、镜像等手段创建面 |
| **②** | 检查自相交 | 自相交会导致网格剖分失败，wavEDA 只能检测无法自动修复 |
| **③** | 构造外包层 | 尺寸不能过大，最好和端口齐平 |
| **④** | 设置端口 | 用矩形包裹端口 → 设激励 → 设模式数 → 激励方向朝外 |
| **⑤** | 赋予材料 | 为各区域指定材料属性 |
| **⑥** | 设置边界条件 | 默认 ABC（吸收边界条件） |

### 2. 仿真配置

| 步骤 | 操作 | 说明 |
|:----:|------|------|
| **①** | 计算区域 | 盒子包裹区域，无间距 |
| **②** | 求解配置 | 频域 → 选择算法（离散/插值/快速）→ 设置频点 |
| **③** | 快照设置 | 如需时谐场图，提前启用快照 |
| **④** | 网格设置 | 3D网格 → 自适应网格 → 波端口网格 |

### 3. 仿真

| 步骤 | 操作 |
|:----:|------|
| **①** | 波端口模式分析（检查 β 和 α） |
| **②** | 验证 → 运行仿真 |

### 4. 后处理

| 步骤 | 操作 |
|:----:|------|
| **①** | 查看 S 参数（端口结果） |
| **②** | 查看快照/时谐场 |
| **③** | 查看远场 |
| **④** | 导出数据 |

## 关键技巧

- **光学元件**：建议用**插值算法**
- **端口模式比对**：先核对相位常数 β 和衰减常数 α 是否与 COMSOL 大致相同
- **网格控制**：θ 越大网格越细
- **时谐场**：在快照设置中启用，导出 GIF
