# 案例教程：Y 分支波导仿真

> 基于 wavEDA 的 Y 分支波导（Y-branch Waveguide）完整仿真流程。

---

## 模型参数

| 项目 | 内容 |
|------|------|
| **模型尺寸** | 16 × 5.6 × 0.4 μm |
| **材料** | Si（εᵣ=11.937）、SiO₂（εᵣ=2.086） |
| **仿真范围** | 187 ~ 200 THz（101 个频点） |
| **激励方式** | 波端口激励 |
| **算法选择** | 插值算法（推荐）/ 离散算法 |

## 仿真流程

### 1. 建模
- 导入几何文件（`.stp` 格式）
- 检查自相交
- 构造外包层（与端口齐平）
- 设置波端口激励（3 个端口）
- 设置材料属性

### 2. 配置
- 求解配置：频域 → 插值算法 → 101 个频点（光学元件推荐插值）
- 网格设置：3D网格 + 自适应网格
- 快照设置：如需查看时谐场，启用快照

### 3. 仿真
- 波端口模式分析（一次性分析所有端口，记录 β 和 α）
- 运行仿真

### 4. 后处理
- 查看 S 参数
- 查看时谐场图
- 导出数据

## 结果数据（插值算法）

| 指标 | 数值 |
|------|:----:|
| 初始网格数量 | 18,890 |
| 最终网格数量 | 338,160（4 次 0.02） |
| 自由度数 | 2,208,774 |
| 自适应网格迭代时间 | 4 min 20 s |
| 仿真时间 | 13 min 28 s |
| **相位常数 β** | 1.008 × 10⁷ |
| **衰减常数 α** | 762,400 |

## 算法对比

| 算法 | 仿真时间 | 适用场景 |
|:----:|:--------:|----------|
| **插值** | 13 min 28 s | 推荐，效率高 |
| **离散** | 2 h 37 min | 高精度需求 |

## 已知问题

- ⚠️ 能量无法有效传导到 2、3 端口，与 EastWave（FDTD）结果差异较大
- 可能与波导结构的边界条件或端口设置有关，需进一步排查
