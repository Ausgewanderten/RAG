# 边界条件

> wavEDA 中边界条件的设置方法与说明。

---

## 默认边界

- **默认 ABC（吸收边界条件）**，模拟电磁波向无限远辐射
- 默认贴近材料，即边界内材料延伸

## 边界条件类型

| 类型 | 说明 |
|------|------|
| **ABC** | 吸收边界条件（默认），适用于辐射问题 |
| **PBC** | 周期边界条件，适用于周期性结构 |
| **边界材料设置** | 可手动指定边界处材料属性 |

## 注意事项

- 无限边界默认为 ABC，默认贴近材料
- 若需更改边界材料，需在边界条件设置中手动指定
- 对比 COMSOL：wavEDA 用 ABC，COMSOL 用 PML
