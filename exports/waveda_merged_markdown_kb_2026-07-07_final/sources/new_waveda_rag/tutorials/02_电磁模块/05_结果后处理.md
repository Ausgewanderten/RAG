# 结果后处理

> wavEDA 仿真完成后查看和分析结果的方法。

---

## S 参数查看

| 操作 | 路径 |
|------|------|
| **查看 S 参数** | 后处理 → 端口结果 → 选择 S 参数 |
| **新建端口结果** | 电磁结果 → 右键端口 → 新建端口结果 |
| **导出数据** | 端口节点下有数据后可右键导出 |

> ⚠️ wavEDA 无需手动开关端口，跑一次自动调端口生成多端口 S 参数数据。

## 时谐场 / 快照查看

| 操作 | 路径 |
|------|------|
| **快照设置** | 快照 → 快照设置 → 启用 → 选择频率 → 应用 |
| **面快照** | 右键新建面快照 → 设置形状、快照区域 |
| **查看结果** | 电磁结果 → 右键快照 → 新建快照结果 → 选择参数 → 添加 → 完成 |
| **导出 GIF** | 快照查看时右上侧勾选时谐场，停止后可右键导出 |

## 远场查看

| 操作 | 路径 |
|------|------|
| **查看远场** | 右键远场 → 新建 3D 远场结果 → 选择频率 |
| **参数选择** | 选择增益 / E / H / 方向性 / 轴比 |
| **单位调整** | 远场增益单位可在后处理中调整（如 dBi / dBic） |

## 网格查看

- 查看网格划分质量
- 查看最终网格数量和自由度

## 关键数据记录

仿真完成后建议记录以下数据：

- 网格数量
- 自由度数
- 插值点数
- 自适应网格迭代时间
- 仿真时间
- 初始网格数量
