# 端口设置

> wavEDA 中端口与激励的设置方法及注意事项。

---

## 集总端口（Lumped Port）

- 适用于射频/微波器件
- 操作：先建立面 → 在面上建立集总端口
- ⚠️ **扫参时端口位置也要参数化**，防止扫参时端口脱离

## 波端口（Wave Port）

- 适用于光电子/波导器件
- 操作：用矩形包裹端口 → 设置模式数 → 设置激励方向（朝外）

## 端口模式分析

- **操作路径**：仿真 → 波端口模式分析（一次性分析所有端口）
- **单端口查看**：电磁 → 激励 → 波端口 → 选择端口右键显示模式
- **重点关注**：**相位常数 β** 和 **衰减常数 α**
- **用途**：确认两款软件选择的端口模式一致（β 和 α 数值应大致相同）
- **模式选择**：可选择不同模式查看

## 端口网格

- 波端口网格设置方式与 3D 网格相似
- 用于端口模式分析的精度控制

## 多端口处理

- wavEDA **无需手动开关端口**，跑一次自动调端口生成多端口 S 参数数据
- 对比 COMSOL 需手动配置激励源端口

## S 参数导出

- 端口结果 → 选择参数类型（S 参数）→ 选择单位 → 选择具体参数 → 添加曲线 → 应用
- 电磁结果 → 右键端口 → 新建端口结果
- 端口节点下有数据后可右键导出
