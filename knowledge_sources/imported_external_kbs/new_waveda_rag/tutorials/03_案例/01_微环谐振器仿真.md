# 案例教程：微环谐振器仿真

> 基于 wavEDA 的微环谐振器（Micro-ring Resonator）完整仿真流程。

---

## 模型参数

| 项目 | 内容 |
|------|------|
| **模型尺寸** | 14 × 9.474 × 1 μm |
| **材料** | Si（εᵣ=11.937）、SiO₂（εᵣ=2.086） |
| **仿真范围** | 187 ~ 200 THz（101 个频点） |
| **激励方式** | 波端口激励 |
| **算法选择** | 离散算法 |

## 仿真流程

### 1. 建模
- 导入几何文件
- 检查自相交
- 构造外包层
- 设置波端口激励
- 设置材料属性

### 2. 配置
- 求解配置：频域 → 离散算法 → 101 个频点
- 网格设置：3D网格 + 自适应网格（θ=0.02, 4次迭代）
- 快照设置：启用快照，选择频率

### 3. 仿真
- 波端口模式分析（记录 β 和 α）
- 运行仿真

### 4. 后处理
- 查看 S 参数
- 查看时谐场图（GIF）
- 导出数据

## 结果数据

| 指标 | 数值 |
|------|:----:|
| 网格数量 | 158,865（4 次 0.02） |
| 自由度数 | 1,031,850 |
| 自适应网格迭代时间 | 2 min 38 s |
| 仿真时间 | 55 min 9 s |
| **相位常数 β** | 8.2438 × 10⁶ |
| **衰减常数 α** | 20,040 |

## 关键要点

- 使用**离散算法**以获得高精度结果
- 模式分析时重点关注 β 和 α，与 COMSOL 比对确认模式一致
- 电场图使用**时谐场动图（GIF）** 展示
- S 参数计算用复数域误差公式（含极性翻转检测）
